### 사용법

$ git clone "https://github.com/LIkelion-at-KOREATECH/Naro.git"

$ pip install django

$ pip install pillow

$ python manage.py migrate

$ python manage.py runserver

### 본인 브랜치 명

 - 임우열 : W_branch
 - 배수빈 : B_branch
 - 문찬웅 : M_branch
 - developer : mater 배포 전 점검 branch

 ### 개발 기능

 - 임우열 : 회원가입 및 로그인
 - 배수빈 : main_bf_login, main_af_login (메인 화면)
 - 문찬웅 : Css 및 Design
